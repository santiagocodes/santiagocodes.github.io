Weather App

El proyecto original empezó como una página web donde se utilizaban dos APIs para obtener datos sobre el clima. Según avancé, decidí expandir el proyecto por lo que añadí otras funcionalidades como cambiar la imagen a una que haga referencia al clima actual, un botón para dark-mode y la utilización de tailwindCSS para el diseño.

Tecnologías utilizadas: NodeJS, HandleBars, TailwindCSS, Heroku y los APIs de WeatherStack y MapBox.

EarnEat Dashboard

Dashboard totalmente responsivo, creado para el uso de los empleados de ApetEat. Trabajé con un equipo de tres en donde todas compartiamos tareas tanto de front-end como de back-end, pero yo era la encargada de que todo saliera bien en el front-end.

Tecnologías utilizadas: MySQL, NodeJS, ReactJS, SemanticUI, ExpressJS, PassportJS, entre otros.

Random Quote Generator

Esta página selecciona citas de Los Simpsons (con su propia imagen) de forma aleatoria. La información es obtenida de una API al hacer click en el botón.

Tecnologías utilizadas: Axios, ReactJS, CSS y el API thesimpsonsquoteapi.glitch.me/.
