Preguntas

Tal vez tengas algunas dudas sobre mí. En esta sección podrás ver algunas contestaciones que podrían satisfacer tu curiosidad.

  • ¿Qué herramientas utiliza a diario?

    Mi editor de elección es Visual Studio Code. En este utilizo varias extensiones como Emmet, ES7 React/Redux/GraphQL/React-Naive snippets, Live Server y Live Share.

    En cuando a navegadores, usualmente utilizo Chrome. En este tengo varias extensiones para diferentes objetivos. Daily 2.0 es una de las herramientas que utilizo para surtirme de noticias sobre el desarrollo web. Eye Dropper lo uso para detectar el código exacto de cualquier color que quiera utilizar en el futuro. Wappalyzer me permite descubrir que tecnologías utiliza la página web que este observando en ese momento. Claro está que no se me puede olvidar la consola para inspeccionar los diferentes elementos de mi página, mientras que React Developer Tools cubre las deficiencias que la consola tiene cuando se trabaja con ReactJS.

    También uso otras herramientas (como Materialize o MomentJS), pero ya esas dependen del proyecto y cuales este necesite.

  • ¿Cómo llegó a formarse en Web Development?

    Mi formación relacionada al mundo de la programación empezó mientras estudiaba la carrera en marketing con un curso básico de C++. Es ahí donde mi interés por este tipo de materias comenzo.

    Trabajando en marketing siempre ves algo, aunque sea básico, de desarrollo web, pero no es hasta que solicité una beca en Wild Code School que mi formación formal en desarrollo tuvo sus inicios. Después de varias entrevistas por Skype con los headquarters en Francia y una prueba en JavaScript, gané una beca completa para su programa de Full-Stack Web Development.

    Al graduarme del Bootcamp con Wild Code School me dedique a practicar todo lo que habia aprendido y a expandir mis conocimientos al, por ejemplo, obtener diferentes certificaciones de freeCodeCamp y siguiendo tutoriales en youtube de grandes programadores como The Net Ninja y Traversy Media.
